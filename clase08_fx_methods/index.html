Funciones JS

  • Es un conjunto de instrucciones agrupadas. Tienen una finalidad en particular y permiten la reutilización.

    * Ver código funcional en archivo ejercicio-funciones.js

  • La función flecha es una alternativa compacta a una expresión de función tradicional, pero es limitada y no se puede utilizar en todas las situaciones. Diferencias y limitaciones: No tiene sus propios enlaces a this o super y no se debe usar como métodos. No tiene argumentos o palabras clave new.

    * Ver código funcional en archivo ejercicio-funciones.js

  • Si una arrow function posee una única instrucción, se puede utilizar un retorno implísito:.

    * Ver código funcional en archivo ejercicio-funciones.js

Método map

  • El método map nos permite iterar por cada elemento de un determinado array y nos devolverá otro array luego de haber procesado las instrucciones que le pedimos. Se utiliza para hacer operaciones, como sumar valores, multiplicar, agregar key a un objeto, etc

    * Ver código funcional en archivo ejercicio-funciones.js

  • Ejemplo comparativo entre map y for para que devuelvan el nombre de cada objeto dentro del array
